The file path prefix that should NOT be created in Azure Storage.
  • Specify this relative to the workspace to match the 'List of Artifacts to upload'
  • Jenkins environment variables can be used in this path

For example, given -
Parameter Value
List of Artifacts to upload target/deployment/binaries/**/
Remove prefix target/deployment/

the upload would only contain the 'binaries' folder in Azure Storage, not target/deployment/binaries.
